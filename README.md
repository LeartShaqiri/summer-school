# Summer School — AI & ML

A modern single-page website exploring AI and ML with **Hermes Agent** and **Claude AI**.

Built with vanilla HTML, CSS, and JavaScript — no frameworks, no bloat.

## Project Structure

```
summer-school/
├── index.html          # Main page — 4 stacked sections
├── css/
│   └── style.css       # Design system & all styles
├── js/
│   └── main.js         # Animations & interactions
└── assets/             # Images, fonts, etc.
```

## Design

- **Palette**: Deep dark backgrounds with electric blue/violet accents
- **Typography**: Inter (sans-serif) + JetBrains Mono (code)
- **Effects**: Scroll reveals, glassmorphism cards, ambient gradient glow, parallax mouse tracking
- **Four sections** stacked vertically, each a full viewport

## Sections

1. **Hero** — overview / headline
2. **Learn** — about AI & ML
3. **Tools** — Hermes Agent + Claude AI deep dive
4. **CTA** — call to action / footer

## Getting Started

Open `index.html` in any browser, or serve locally:

```bash
python -m http.server 8080
# → http://localhost:8080
```
